# ИТ для филологов — образовательный портал

Vercel-ready Next.js приложение с материалами курса «ИТ для филологов». На главной странице — быстрый доступ к лекциям, практикам, презентациям, силлабусу, видео и команде авторов. Для каждой категории предусмотрена отдельная страница с кратким описанием и списком ключевых материалов.

Особенности:

- Двуязычный интерфейс (русский и узбекский) с мгновенным переключателем языка.
- Два интерактивных задания: по аппаратной настройке и по операционным системам с офисными приложениями.
- Современный визуальный стиль с акцентом на читабельность и адаптивность.

## Технологии

- [Next.js 14](https://nextjs.org/) (App Router, TypeScript)
- React 18
- CSS Modules / глобальные стили без фреймворков

## Локальный запуск

> Требуется Node.js 18.17 или выше.

```powershell
npm install
npm run dev
```

После запуска приложение будет доступно по адресу <http://localhost:3000>.

## Сборка и деплой

```powershell
npm run build
npm start
```

Для деплоя на Vercel импортируйте репозиторий и выберите фреймворк **Next.js**. Все параметры сборки будут распознаны автоматически.

## Структура

- `app/page.tsx` — главная посадочная страница (обёртка над клиентским компонентом с поддержкой переключения языка)
- `app/(sections)/[slug]/page.tsx` — странички разделов (лекция, практика и т.д.) с локализованным контентом
- `components/` — клиентские компоненты (языковой контекст, переключатель, интерактивное задание)
- `lib/translations.ts` — текстовые ресурсы на русском и узбекском языках
- `app/globals.css` — глобальные стили оформления
- `public/` — директория для изображений и других статических ресурсов (добавьте по необходимости)

## Дальнейшее развитие

- Добавьте реальные материалы (PDF, презентации, видео) в отдельные разделы.
